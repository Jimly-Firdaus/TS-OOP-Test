# Payment Gateway Service Simulation
Object Oriented Programming using Typescript

# How to Run
## Clean installation
On terminal, do this
```shell
npm ci
```
After installation, on the `src` dir, do this
1. Compile `index.ts`
```bash
tsc .\index.ts
```
2. Run the compile result (.js file)
```bash
node .\index.js
```
